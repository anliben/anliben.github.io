# Regras para Agentes de IA

Este arquivo contém regras permanentes que devem ser seguidas por todos os agentes de IA que trabalham neste projeto.

## Regras de Escrita em Português

### Acentuação Obrigatória

**REGRA CRÍTICA:** Todo o conteúdo deste blog é escrito em português brasileiro. É **obrigatório** utilizar acentuação correta em todas as palavras.

Exemplos de palavras que devem ser acentuadas corretamente:

- voce → **você** (com acento agudo no e)
- nao → **não** (com til no a)
- cafe → **café** (com acento agudo no e)
- esta → **está** (com acento agudo no a, quando verbo)
- e → **é** (com acento agudo, quando verbo ser)
- unica → **única** (com acento agudo no u)
- cerebro → **cérebro** (com acento agudo no e)
- visao → **visão** (com til no a)
- proposito → **propósito** (com acento agudo no primeiro o)
- reflexao → **reflexão** (com til no a)
- reflexoes → **reflexões** (com til no o)
- direcao → **direção** (com til no a)
- especifico → **específico** (com acento agudo no i)
- resistencia → **resistência** (com acento circumflexo no e)
- toxico → **tóxico** (com acento agudo no o)
- invisivel → **invisível** (com acento agudo no i)

### Por que isso é importante?

1. **Legibilidade:** Textos sem acentuação são mais difíceis de ler
2. **Profissionalismo:** Acentuação correta demonstra cuidado com o conteúdo
3. **Significado:** Algumas palavras mudam de significado sem acento (ex: "e" vs "é", "esta" vs "está")
4. **Respeito ao idioma:** O português é uma língua rica e suas acentuações fazem parte da sua identidade

### Checklist antes de publicar

- [ ] Verificar todos os acentos agudos (á, é, í, ó, ú)
- [ ] Verificar todos os acentos circunflexos (â, ê, ô)
- [ ] Verificar todos os tils (ã, õ)
- [ ] Verificar cedilhas (ç)
- [ ] Revisar o texto completo antes de fazer commit
